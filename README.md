# Reegle ジラフユニット 商談プレゼンシステム v4
## 「Lacquer & Vermillion / 漆と朱」

オンライン商談（Zoom/Teams等）で台本を見ながら投影スライドを進めるシステム。
3タイプ（自治体／幼稚園／写真スタジオ）切替対応。

## ファイル
- present.html — 進行用（自分用、台本＋プレビュー＋切替）
- index.html — 投影用（先方に見せる、スライドのみ）

## デザイン v4
- 緑×朱（vermillion）の和モダン配色
- ローマ数字（I〜XII）の章マーク、雑誌風エディトリアル構造
- 巨大装飾タイポグラフィ（USPの朱色8.5cqw数字）
- フルブリード写真（カバー＋実績紹介）
- BIZ UDPGothic ＋ Noto Serif JP ＋ Bodoni Moda の3書体構成

## 使い方
1. present.html をブラウザで開く（Chrome/Edge推奨）
2. 上部のタイプ切替で対象を選ぶ
3. 右上の「🖥 投影画面を別タブで開く」をクリック
4. Zoomで**その別タブだけ**を画面共有
5. 手元で台本クリックor←→キーで進める
